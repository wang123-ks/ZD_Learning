# vue/cli的安装和运行
### 步骤一、准备node.js和npm，并确认电脑是否安装成功。

``` sh
node -v
```

``` sh
npm -v
```

### 步骤二、全局安装vue/cli
#### 1.安装命令

``` js
npm install -g @vue/cli  // -g为安装到全局
```

#### 2.测试是否安装成功

``` sh
vue -V
```

### 步骤三、运行：用cmd进到对应文件夹，在该路径下运行serve

``` sh
npm install
```

``` sh
npm run serve
```

### 步骤四、展示
上述文件下的项目会运行在8080端口，打开浏览器输入<http://localhost:8080>就可以看到效果。若8080端口被其他项目占用了，vue会使用其他端口。
